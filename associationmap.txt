1. NP + 'be' + ADJP >> JJ IS-PROPERTY-OF NN
  milk is white >> white IS-PROPERTY-OF milk

2. NP + 'be' + NP >> NN IS-A NN
  a dog is an animal >> dog IS-A animal

3. NP containing JJ + NN >> JJ IS-PROPERTY-OF NN
  the big house >> big IS-PROPERTY-OF house

4. VP containing RB + VB >> RB IS-PROPERTY-OF VB
  It quickly moves >> quickly IS-PROPERTY-OF moves

5. VP + ADJP >> RB IS-PROPERTY-OF VB
  Cats can run fast >> fast IS-PROPERTY-OF run

9. sbj + VB + RB >> NN, VB (how) RB

10. VB + obj >> NN IS-OBJECT-OF VB
  This button releases the hounds. >> hound IS-OBJECT-OF release

11. VP + PNP >> VB,NN (preposition) PP
  Look at your hand. >> look, hand (preposition) at
  Look behind the door. >> look, door (preposition) behind
